Under the hood

How the model thinks.

01
Three years of ball-by-ball data

Cricsheet IPL 2023 → 2026. Every delivery, every wicket, every toss — 60k+ deliveries across 250+ matches.

02
Sixteen leak-free features

Rolling form, venue win-rate, H2H, scoring & conceding rates, toss history — computed using only data from before each match.

03
Gradient-boosted classifier

Calibrated probability for the winner. Score & boundaries blend team trends with venue norms.

04
Walk-forward validation

No future leakage. Every back-tested match is predicted using only the past. The hit rate is what we'd actually have hit, live.
